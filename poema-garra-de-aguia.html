鹰 爪 擒 拿 18 要 诀

Este poema é a chave para a compreensão do sistema Garra de Águia.

O Sistema Garra de Águia tem um poema de 18 caracteres que é a chave para a compreensão deste método. Os alunos devem memorizar este poema e se esforçar para entender cada palavra profundamente. Este poema abre os princípios fundamentais por trás do Chin Na. Uma vez que você compreender totalmente esses caracteres, você poderá aplicá-los a qualquer momento para sua prática.


1. Dian 点- Local, Ponto
Para atacar precisamente os pontos de pressão ou acupontos.

2. Na 拿 - Agarrar
Para agarrar as articulações, tendões e pontos.

3. Suo 锁 - Bloquear, Travar
Para aplicar chave na garganta, articulações e pernas.

4. Kou 扣 - Deter Para deter a perna ou o cotovelo.

5. Ban 搬 - Tira, mover, remover
Para mover o adversário.

6. Chan 缠 - torcer , girar
Para amarrar como uma videira, atacando o pulso ou os tornozelos.

7. Qie 切 - Corte
Para atacar como uma serra ou fatiar, como o desenho de um corte.

8. Ya 压 - Pressionar
Para pressionar o cotovelo e pernas.

9. Sui 摔 - Jogar
Para lançar o seu adversário. (A Sui do Shuai Chiao)

10. Chin 擒 - capturar
Para pegar a cabeça, braços ou pernas

11. Zhua 抓 - Agarrar, prender, garra
Para atacar com garra.

12. Ba 拔 - Puxe, puxe para cima
Para puxar os membros.

13. Ning 拧 - Prender, torça
Para prender os braços e pulsos.

14. Feng 封 - Fechar, bloquear
Para bloqueio da garganta ou das pernas

15. Bi 闭 - Prender, fechar
Para parar a respiração pela garganta

16. Die 跌- Queda, derrubar
Para fazer com que seu oponente caia. Diferente do Sui, porque o oponente cai para baixo em vez de ser jogado.

17. Lan 拦 - Pare de entrar
Para afastar.

18. Diao 刁 - Astucioso, hábil, esperto
Para ser capaz de reverter uma garra. Esta técnica é mutável, e não apenas fixa, dependendo de como o adversário vem.
Usa Na e Ko
